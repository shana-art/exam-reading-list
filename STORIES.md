# User stories – Läslistan

## US1: Navigera mellan vyer
Som besökare vill jag kunna navigera mellan katalog, favoriter och lägga-till
så att jag kan använda sidan.

## US2: Favoritmarkera bok
Som besökare vill jag kunna favoritmarkera en bok
så att jag kan bygga min läslista.

## US3: Ta bort favorit
Som besökare vill jag kunna ta bort en bok från favoriter
så att listan blir korrekt.

## US4: Lägga till bok
Som besökare vill jag kunna lägga till en ny bok
så att katalogen kan växa.
